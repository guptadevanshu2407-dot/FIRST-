# Expense Tracker System

A simple Expense Tracker web application built using:

- Python
- Flask
- HTML
- CSS
- JavaScript

This project allows users to:

- Add expenses
- View all expenses
- Delete all expenses

---

# Installation

## Step 1: Clone Repository

```bash
git clone <your-github-repo-link>
```

---

## Step 2: Open Project Folder

```bash
cd expense-tracker
```

---

## Step 3: Install Flask

```bash
pip install flask
```

---

## Step 4: Run Application

```bash
python app.py
```

---
